# Ankit Vashisth | Portfolio 🚀

A highly interactive, dynamic, and premium personal portfolio built with **React** and **Vite**. This project serves as a showcase of my skills, experience, and projects, featuring cutting-edge web design aesthetics like glassmorphism, 3D hover effects, and continuous dynamic animations.

## ✨ Features

- **Interactive 3D Hero Section:** The profile picture tracks mouse movements and tilts in 3D space, while glowing technology bubbles dynamically orbit the portrait like satellites.
- **Dynamic Background Engine:** The background seamlessly cross-fades into 5 distinct environments as you scroll through different sections:
  - 🌌 **Hero:** Deep space starry background using particles.
  - 🏙️ **Experience:** Cyberpunk neon grid.
  - 🌊 **Projects:** Fluid, glowing liquid waves.
  - 🔺 **Skills:** Floating geometric polygons.
  - 🌌 **Education:** Shimmering Aurora Borealis gradients.
- **Premium Typography:** Utilizes beautiful, modern Google Fonts (*Space Grotesk* for headings and *Outfit* for clean body text).
- **Responsive Design:** Fully optimized for mobile, tablet, and desktop devices.
- **Glassmorphism UI:** Sections and cards are styled with frosted glass effects to blend beautifully with the dynamic backgrounds.

## 🛠️ Tech Stack

- **Framework:** React.js (via Vite for blazing-fast builds)
- **Styling:** Vanilla CSS3 with advanced CSS animations, keyframes, and CSS variables.
- **Animations:** Custom `@keyframes` (orbiting physics, pulsing neon glows, 3D preserves).
- **Icons:** `lucide-react` & `react-icons`.
- **Particles Engine:** `tsparticles` for the interactive space background.

## 🚀 Getting Started

To run this project locally on your machine:

1. **Clone the repository:**
   ```bash
   git clone https://github.com/Ankitji579/My_Portfolio.git
   cd My_Portfolio
   ```

2. **Install dependencies:**
   ```bash
   npm install
   ```

3. **Start the development server:**
   ```bash
   npm run dev
   ```

4. **Build for production:**
   ```bash
   npm run build
   ```
